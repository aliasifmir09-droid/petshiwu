## 🎯 **Technical Details**

### **Image Specifications:**
- **Source**: Pexels.com
- **Format**: JPEG
- **Size**: 800x800px
- **Compression**: TinyPNG compression
- **Quality**: High (q=90 equivalent)

### **Image URL Structure:**
```
https://images.pexels.com/photos/{PHOTO_ID}/pexels-photo-{PHOTO_ID}.jpeg?auto=compress&cs=tinysrgb&w=800
```

### **Parameters:**
- `auto=compress` - Automatic compression
- `cs=tinysrgb` - Color space (sRGB)
- `w=800` - Width in pixels
